# Job Application Tracker (Chrome Extension)

A lightweight Chrome extension that helps you track your job applications while browsing.  
Add roles you’re applying for, keep notes, update statuses, and manage everything directly from your browser — without needing a backend or external accounts.

---

## Overview

This extension provides a clean, simple interface for logging job applications:

- Track job titles, companies, links, statuses, and notes  
- Edit or delete entries anytime  
- Stored locally using Chrome storage — **your data stays on your device**  
- Super fast, fully client-side, no backend required  

Perfect for staying organised during an active job hunt.

---

## Technology Used

- **HTML**
- **CSS**
- **JavaScript**
- **Chrome Extension Manifest v3**
- **Chrome Storage API** for saving data

No external server or database is required.
